Sistema de gerenciamento de hotel desenvolvido em Java, com execução em terminal.
O projeto simula o controle de hospedagens, permitindo organizar reservas, hóspedes e consumo durante a estadia, utilizando lógica implementada manualmente com arrays e matrizes.

🛎 Reserva de quartos
❌ Cancelamento de reservas
📋 Listagem de quartos ocupados
🔍 Consulta de hóspedes por quarto
✏️ Edição de dados do hóspede
🧃 Registro de consumo no frigobar
💸 Cálculo de check-out com valor total

##  ⋆˚࿔ 𝗙𝘂𝗻𝗰𝗶𝗼𝗻𝗮𝗺𝗲𝗻𝘁𝗼
O sistema funciona através de um menu interativo no terminal, onde o usuário escolhe as opções digitando números.
As informações são armazenadas em arrays, permitindo controlar até 100 hóspedes e quartos.
O frigobar é gerenciado por uma matriz, registrando a quantidade de produtos consumidos em cada quarto.



##  ✌︎㋡ 𝗟ó𝗴𝗶𝗰𝗮
A aplicação utiliza estruturas condicionais e de repetição para validar entradas e controlar o fluxo do programa.
Os dados são manipulados diretamente em memória durante a execução, sem uso de banco de dados.
